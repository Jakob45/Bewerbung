Bilder einfügen auf Webseiten

Zum Einfügen eines Bildes wird der Tag <img> mit folgenden Attributen verwendet:

  • src gibt an, wo sich das Bild befindet.
  • alt zeigt einen alternativen Text an, falls das Bild falsch verlinkt ist oder am ursprünglichen Ort nicht mehr vorhanden ist ("broken link").
  • width gibt die Breite des Bildes in Pixel an.
  • height gibt die Höhe des Bildes in Pixel an.
  • title zeigt einen Text an, wenn man den Mauszeiger über das Bild bewegt.

Man kann auch Bilder direkt aus dem Internet einfügen, aber Achtung!

  • Bilder aus dem Internet sind meistens urheberrechtlich geschützt und dürfen nicht ohne ausdrückliche Erlaubnis auf der eigenen Website veröffentlicht werden.
  • Das Bild könnte auf dem Server verschoben oder gelöscht werden und ist dann auf meiner Seite nicht mehr zu sehen ("broken link").
  • Das Laden des Bildes von einem anderen Server könnte langsam sein.

Links einfügen auf Webseiten

Zum Einfügen eines Links wird der Tag <a> verwendet. Mit dem Attribut href wird auf den Pfad zur verlinkten Seite verwiesen.
Bei Links zu fremden Seiten sollte man noch das Attribut target mit dem Wert "_blank" verwenden, damit die Seite in einem neuen Tab angezeigt wird.
